feat(android): unified design system & metadata enrichment for blocker and scheduler modules

This commit implements a comprehensive UI/UX overhaul for the native Android enforcement 
and notification layers, ensuring brand consistency and improved user context during 
active commitments.

Blocker Module Enhancements:
- Data Layer: Enhanced BlockerAccessibilityService to query the SQLite vault for 
  'task_instances' metadata. It now extracts 'title' and 'end_time' to provide 
  violation-specific context.
- UI Redesign: Refactored BlockerOverlayActivity to a premium "distraction-free" 
  aesthetic:
    * Pure black (#000000) background for OLED efficiency and focus.
    * Hierarchical typography using 'sans-serif-light' for status (letter-spaced), 
      'sans-serif-medium' for titles, and bold variants for actions.
    * Standardized 56dp pill-shaped 'Primary Blue' (#4FA0FF) action buttons.
    * Integrated localized "Active until HH:mm" expiration indicator.
- Resilience: Fixed structural corruptions and stabilized the enforcement pulse logic.

Scheduler Module Enhancements:
- UI Synchronization: Rewrote AlarmActivity's programmatic layout to perfectly match 
  the blocker module's design language.
- Aesthetics: Migrated from flat layouts to a refined design system:
    * Removed all emojis for a professional, clinical appearance.
    * Transitioned from raw pixel measurements to dynamic dp-based scaling.
    * Implemented a distinct status label (e.g., "UPCOMING", "TIME'S UP") positioned 
      above the task title with 0.2f letter spacing.
- Button Standardization: Updated the 'Dismiss' button to a GradientDrawable pill 
  shape, consistent with the blocker's 'Return to CommitT' button.

Technical Improvements:
- Unified typography tokens (sans-serif-light/medium/bold) across all native components.
- Standardized padding and margin constants via internal dp-resolution helpers.
- Fixed structural regression in BlockerAccessibilityService.kt involving misplaced 
  imports and stray expressions.
