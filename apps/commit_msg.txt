style: theme Location Picker, geofences, and shared status indicators

Unified the location selection flow and shared verification components with the Saffron design system tokens.

Key Changes:
- Theme Expansion: Added `surfaceLight` (subtle overlay) token to `theme.ts` for enhanced depth on status badges and popovers.
- Location Modal: Refactored `LocationPresetPickerModal` to use `THEME.colors.surface` and updated Google Static Maps API logic to inject Saffron (`THEME.colors.primary`) for geofence paths and markers.
- Status Indicator: Overhauled `VerificationStatusCircle` with the new Saffron palette, thickened borders (1.5px), and restored tactile depth using the `surfaceLight` background token.
- Popover Menu: Tokenized `ActionMenu` by mapping its floating container to `surfaceElevated`, adding a crisp `surfaceLight` border, and syncing destructive actions with `THEME.colors.danger`.
- Global UI: Replaced hardcoded grays and whites across location list items with `textMain` and `textMuted` tokens.

This ensures that modal pickers and status badges maintain a premium, consistent HUD aesthetic throughout the commitment creation flow.
