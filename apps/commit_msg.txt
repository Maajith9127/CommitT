feat(native): implement premium sliding tab transitions for Presets and Choose screens

Replaced the static tab-switching logic with a high-performance horizontal 
paging system, providing a fluid, native "slide" feel across the app.

- TabsBar: Added an animated sliding indicator (the blue bar) using Reanimated 
  `withTiming` (180ms) for a quick, snappy snap that visually leads the page transition.
- Dynamic Scaling: Implemented `onLayout` tracking in the `TabsBar` to 
  automatically calculate indicator width and travel distance, ensuring 
  perfect alignment on any device.
- Presets Hub: Refactored the main Presets screen to use a horizontal paging 
  container, linking the top bar selection to the scroll position.
- Choose Screen: Integrated paging into the blocklist configuration flow, 
  keeping the virtualized Apps list performant while adding slide 
  transitions for Webs and AI tabs.
- Scroll Sync: Implemented `onMomentumScrollEnd` to ensure manual swipe 
  gestures keep the UI state and header titles perfectly in sync.
