feat(presets): implement keyboard-aware editing and quick-add FAB

Enhances the preset management experience with better ergonomics, stable 
map interactions, and an expressive creation workflow.

- Keyboard-Aware UI: Implemented panel elevation in the location editor using 
  Reanimated to keep inputs fully visible and interactive above the keyboard.
- Floating Add Button (FAB): Added a premium circular '+' button to the 
  Presets Hub with a pre-seeding logic that clones the 'latest' location 
  data for fast duplication and refinement.
- Dual Save/Create Logic: Updated the editor and backend to support both 
  in-place refinements (update) and brand-new entries (create) depending 
  on the entry point.
- Stable Map Interactions: 
    - Disabled automatic camera re-centering during manual taps/long-presses.
    - Prevented keyboard dismissal during map interaction (scroll/zoom/pin-drop).
- UI/UX Polish: 
    - Replaced generic icons with white MaterialCommunityIcons.
    - Integrated a unified name input field with a 'close-circle' dismissal button.
    - Added createLocationPreset mutation to the Convex backend with 
      automatic usage signal initialization.
